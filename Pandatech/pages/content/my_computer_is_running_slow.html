My computer is running slow or it freezes

First, you need to identify if this is a software or hardware related issue. The hardware-related issues/solutions are not limited to:

  • Overheating – try cleaning up the fans and heat sinks inside of the computer case including the ones for the CPU, video card, and power supply.
  • Insufficient amount of RAM – add more RAM or replace the current one with a larger memory chip.
  • Slow or fragmented hard drive – you may need to upgrade or defragment your hard drive.
  • Slow video card or CPU – you may need to upgrade these components.
The software solutions could include the following:
1. Uninstall any unnecessary programs by:
  1. Click on the Windows “Start” button
  2. Click on “Control Panel”
  3. Click on “Programs and Features”
  4. Click on any program you want to remove and, at the top, click “Uninstall”. Read the prompts and uninstall the program.
2. Move any large files from the Desktop folder into a different folder on the hard drive.
  1. Click on the Windows “Start” button.
  2. Click “Computer”.
  3. Find the folder where you want to relocate the files.
  4. Click and drag the large files from the Desktop over to the folder you have chosen.
3. Stop any already started unnecessary programs and prevent them from starting every time when Windows starts.
  • In the settings of a specific program, if there is an option, uncheck “Start when my computer starts” or check “Don't start when my computer starts.”
  1. Hold the Windows key and press “R” at the same time. This opens the “Run” command.
  2. Type in "msconfig" and press “Enter”. A “System Configuration” window will open.
  3. Go to the “Start Up” tab and uncheck the unnecessary programs you don't want to start automatically every time the operating system starts.
  4. Click “Apply” and “Ok”. Then close all programs and click “Restart” when the window prompts this.
4. Keep your operating system up to date.
  1. Click on the Windows “Start” button
  2. Click “Control Panel”
  3. Click “Windows Update”
  4. Install the necessary updates
5. Keep your anti-virus or anti-malware protection up-to-date.
6. Low virtual memory:
  1. Increase the amount of virtual memory by clicking on “Start”
  2. Type in the search field “Advanced System Settings”
  3. On the System Properties window go to“Advanced” tab
  4. “Performance Settings”
  5. “Advanced” tab on Performance Options
  6. “Virtual Memory Change”
  7. Click on “Custom size”
  8. Type the recommended amount. Click “Apply” and restart the computer.
If none of this works, seek professonal help (like mine).
